# chronicallyonline
En esta web final, me propuse plasmar la nostalgia hacia la Internet "de antes", no con el ánimo de romantizar el pasado, sino con la intención de rescatar aquellas estéticas, actividades y contenidos que nos unían como comunidad. Quise regresar a esa web que me tocó, con menos brainrot hecho con IA, menos publicidad (o que se camuflaba mejor), menos presión del algoritmo y doomscrolling. Mi deseo es recuperar un pasado utópico y soñar por una Internet más descentralizada y democrática y menos al servicio de unos pocos.
